### 下面介绍怎么在本地搭建自己的wordpress站点 :smile:
在本地搭建wordpress你需要：
- [ ] 下载xampp或者usbserver来搭建本地的php环境
- [ ] 下载最新版的wordpress
- [ ] 下载对应的模板，来丰富你的网站
- [ ] 购买域名、虚拟主机，来实现在线访问

如果这些你都做到了，那么恭喜你。你将拥有自己的首个博客站点。当然用github pages也不错!
